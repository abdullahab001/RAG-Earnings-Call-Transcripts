# Transcripts Folder

This folder should contain earnings call transcripts in .txt format.

## Where to get transcripts:
```
"I collected earnings call transcripts from publicly available 
financial documents to build a RAG system that helps investors 
analyze quarterly results."
```

## Format:
Files should be named: `CompanyName_Quarter_Year_XX.txt`

Example: `Apple_Q3_2024_01.txt`
